# Introduction

Welcome to CLI Slides! This is a command-line presentation tool that renders markdown files as slides.

# Features

- Parses markdown files
- Each H1 header becomes a slide
- Interactive CLI navigation
- Simple and lightweight

# How It Works

1. Write your presentation in markdown
2. Use H1 (#) for slide titles
3. Run the CLI app with your file
4. Navigate through slides with keyboard
